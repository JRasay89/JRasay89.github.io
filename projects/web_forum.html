ICS 415: Web Forum Project

Description

This was a project for my ICS 415 course and it was done together with another student.

There were a few requirements we had to meet for this project which is listed below.

  • Sign in/ login page and guest support.
  • Uniformly styled and not just plain HTML.
  • Responsive and usable on mobile devices.
  • Navigation section and at least three pages.
  • About page which describes the technology used to create the website.
  • Client side interaction.
  • Store information persistently.

Signed in users can create new topics or post messages in existing topics, while guest users can only view messages.
