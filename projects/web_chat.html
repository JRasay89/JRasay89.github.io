Web Chat

Description

A simple public chat room where users can enter and chat with other users. Users must enter a username that is not currently being use to enter the chat room.

How it works
Users must enter a username that is not currently being use to enter the chat room. Usernames are stored in MySQL database, and Ajax is used to call a PHP script that queries a MySQL database to check if the username already exists.
In the chatroom, messages that are sent is stored in a text file which is read every second using Ajax. Any new messages found in the text file is appended to the chat message is box.
