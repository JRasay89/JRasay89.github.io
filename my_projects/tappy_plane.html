Tappy Plane

(Language: C++)

Project Description

Tappy Plane is a game created using C++ and SDL 1.2 which stands for Simple DirectMedia Layer and it is a library designed to provide low level access to audio, keyboard, mouse, joystick, and graphics hardware via OpenGL and Direct3D.

Demo Video

Gameplay

Game play is very simple, you just have to avoid incoming mountain peaks, and for every mountains avoided you gain a point.
